# FitoPro

FitoPro est une application web front-end (HTML/CSS/JavaScript) pour créer, suivre et optimiser ses entraînements.

## Fonctionnalités MVP
- Création d'un programme d'exercices personnalisé.
- Suivi en temps réel avec minuteur de série.
- Suggestions automatiques d'amélioration selon le volume d'entraînement.
- Historique de progression simplifié (séances, séries, streak).
- Défis motivationnels progressifs.

## Lancer en local
Ouvrez simplement `index.html` dans votre navigateur.

## Données
Les données sont sauvegardées localement via `localStorage` (aucun backend requis).
