Politique de confidentialité (RGPD baseline)

FitoPro stocke vos données localement dans votre navigateur. Les données cloud sont envoyées uniquement si vous activez la synchronisation.

  • Données collectées: entraînements, statistiques, paramètres, données cardio importées.
  • Finalité: suivi de progression sportive.
  • Droits: export, suppression locale, suppression cloud via support/endpoint.
  • Contact: privacy@fitopro.local
